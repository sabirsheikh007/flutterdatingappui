# 💕 Flutter Dating App UI

A modern and beautiful Flutter Dating App UI built with clean architecture and responsive design. This project focuses on creating an attractive user interface for a dating application and can be used as a starter template for Flutter developers.

## ✨ Features

- ❤️ Modern Dating App UI
- 📱 Responsive Design
- 🎨 Beautiful Animations
- 🌙 Clean & Minimal Interface
- 🔥 Reusable Widgets
- 📍 User Profile Screen
- 💬 Chat Screen UI
- 🔍 Discover Screen
- ❤️ Match Screen
- 👤 Authentication Screens
- ⚙️ Easy to Customize

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/sabirsheikh007/flutterdatingappui.git
```

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```
